    How to use Caregos safely

    Start with the public record, then verify the care situation.

    Caregos is built for the first pass of a nursing home review. The site organizes CMS star ratings, inspection history, staffing measures, enforcement records, and correction context so families can see which questions deserve attention before a tour or phone call.

    Do not use a single star score as a final decision. A facility can have a strong overall rating and still show recent deficiencies, staffing pressure, complaint surveys, or enforcement actions that matter for a specific resident. A lower-rated facility may also have corrected old findings or serve a resident need that is not visible in a public dataset.

    The practical workflow is to shortlist facilities by location, compare inspection and staffing signals, read the source dates, then confirm availability, payer rules, care planning, visiting policies, and resident-specific needs directly with the facility and the appropriate state resources.
